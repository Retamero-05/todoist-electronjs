<h3>Como rodar o projeto</h3>

```bash
# Clonar Repositório
$ git clone https://github.com/Retamero-05/todoist-electronjs

# Entrar no repositório
$ cd todoist-electronjs

# Instalar dependências

$ npm install

# Iniciar aplicação

$ npm start



```
